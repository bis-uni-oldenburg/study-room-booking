For a reservation, two users (user 1, user 2) have to login with their personal accounts one after another.

  1. Marking: User 1 logs in, clicks a free plan segment, selects the exact booking duration and types in the login ID of user 2.
  2. Reservation: User 2 logs in, clicks the marked plan segment and transforms the marking via 'Confirm marking' into a reservation.
  3. Cancellation: User 1 or user 2 logs in, clicks the plan segment marked as reserved, and deletes the reservation.
  4. Proof of reservation: Basically, rooms can also be used without prior reservation. However, reservations have priority. Click the plan segment marked as reserved and afterwards 'Print receipt' to be able to proof your reservation on site.
